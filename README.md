# valahigh-site

Static landing page for Valahigh Co LLC. Pure HTML + CSS, no build step.

Live at https://valahigh.us. The .com domain redirects here via Squarespace.

## Deploy

Pushes to `main` auto-deploy via Vercel.

## Edit

Edit `index.html` for content, `styles.css` for design. Commit and push.

## Notes

- Color palette matches Clearifi (paper #FAF8F3 + maroon #6B1E3F)
- Logo is a typography wordmark; replaceable with custom mark from
  Claude Design when generated
- Founder bio is a LinkedIn link, not inline copy
